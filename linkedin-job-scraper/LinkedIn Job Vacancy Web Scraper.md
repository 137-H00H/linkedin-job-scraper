**LinkedIn Job Vacancy Web Scraper**



This project collects and analyses job vacancy data using Python. 

The dataset is built by scraping publicly available LinkedIn job listings 

for Data Analyst roles in the United Kingdom.



The project demonstrates the full workflow from data collection to 

exploratory data analysis and visualisation.



**Tools Used**

\- Python

\- requests

\- BeautifulSoup

\- pandas

\- matplotlib



**Outputs**

\- Cleaned CSV dataset

\- Location-based job analysis

\- Job title frequency analysis
